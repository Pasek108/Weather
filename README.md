# Pogoda
Pogoda w React z wykorzystaniem [OpenWeather OneCall API v2.5](https://openweathermap.org/) i [Geonames API](https://www.geonames.org).

## Funkcje
* Wyświetlanie aktualnej pogody
* Wyświetlanie dziennej pogody (7 dni)
* Zamiana lokalizacji
* Zmiana języka
